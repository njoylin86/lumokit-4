Key brand notes when working in this skill:
- The clinic operates in Swedish; default copy language is Swedish.
- Voice is warm, calm, reassuring — never marketing-y, never clinical-cold.
- Use **Cormorant Garamond** for serif headlines, **Outfit** for sans body/UI.
- Use the eyebrow label (11px, ALL CAPS, +0.22em letter-spacing) — it's the brand's strongest typographic signature.
- Sharp corners (4px radii), soft gray shadows, blush + ink palette only.
- No emoji, no gradients, no purple/blue tech-gradient backgrounds, no AI-slop visual tropes.
